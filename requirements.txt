# System Requirements:
# - Python 3.8 to 3.12 (Python 3.10-3.12 recommended)
# - **IMPORTANT**: Python 3.13 NOT supported yet (gensim incompatible)
# - Terminal with Unicode support for proper display
# - Internet connection for web scraping functionality
#
# Python 3.13 Issue:
# The gensim library (v4.3.x) has not been updated for Python 3.13's C API changes.
# Until gensim releases a Python 3.13-compatible version, you MUST use Python 3.12 or earlier.
# See installation workarounds below for solutions.

# Platform-specific Notes:
# - Windows: Ensure terminal supports Unicode (Windows Terminal recommended)
# - macOS: Built-in Terminal.app works well
# - Linux: Most modern terminal emulators work (gnome-terminal, konsole, etc.)

# Installation Notes:
# 1. Create a virtual environment (recommended):
#    python -m venv venv
#    source venv/bin/activate  # On Windows: venv\Scripts\activate
#
# 2. Install dependencies:
#    pip install -r requirements.txt
#
#    **Python 3.13 Users - REQUIRED WORKAROUND**:
#    gensim is not yet compatible with Python 3.13. Choose one option:
#
#    Option A - Use Python 3.12 (RECOMMENDED - most reliable):
#      # Using pyenv (recommended)
#      pyenv install 3.12.8
#      pyenv local 3.12.8
#      python -m venv venv
#      source venv/bin/activate
#      pip install -r requirements.txt
#
#      # Or using system Python 3.12
#      python3.12 -m venv venv
#      source venv/bin/activate
#      pip install -r requirements.txt
#
#    Option B - Skip v1.9.0 features (gensim not needed for v1.0-1.8):
#      # Install all dependencies except gensim
#      grep -v "gensim" requirements.txt | pip install -r /dev/stdin
#
#      # Topic modeling features (Ctrl+Alt+T) will not work
#      # All other features (v1.0-v1.8) work fine without gensim
#
#    Option C - Wait for gensim 4.4.0+ (check PyPI for updates):
#      # Monitor: https://pypi.org/project/gensim/
#      # Once gensim 4.4.0 or later is released with Python 3.13 support,
#      # this issue will be resolved
#
# 3. Run the application:
#    python scrapetui.py
